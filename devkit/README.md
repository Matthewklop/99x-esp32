# 99x-esp32 — DevKit Edition

**ESP32 DevKit rev1 bare-metal firmware. 160 bytes. No SDK. No Arduino. No ESP-IDF.**

Solid blue LED on GPIO2. WDT fed. Runs forever.

## Quick start

```bash
make flash
```

## Status

- ✅ Blue LED (solid on)
- ✅ WDT feed (runs indefinitely)
- 🔄 UART output (dual-core contention)
- 🔄 GPIO blink loop

## Size

160 bytes.
